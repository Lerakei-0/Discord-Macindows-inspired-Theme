
# Discord-Macindows-inspired-Theme

<p align="center">
Theme for discord in with the same general color scheme the macindows theme uses. 

![/screenshot.png](https://github.com/Lerakei-0/Discord-Macindows-inspired-Theme/blob/main/Screenshot.png?raw=true)

## Installation with betterdiscord or vencord
Simply place the .theme.css file in "%appdata%\BetterDiscord\themes" or "%appdata%\Vencord\themes\"
